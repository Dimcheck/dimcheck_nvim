### About
- Clone the repo and cd into it
- Install [stow](https://www.gnu.org/software/stow/) according to your system distribution
- Run:
 ```
 stow -v -t $HOME nvim
 ```
- Install [vim-plug](https://github.com/junegunn/vim-plug)
- Open nvim and run :PlugInstall
- Reopen nvim
